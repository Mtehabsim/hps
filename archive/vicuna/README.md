# vicuna — Vicuna-13B cross-model scripts

Cross-model checks on Vicuna-13B (the project later focused on Llama-3-8B). Extraction, diagnostics,
and robustness/overfitting/param-sweep tests specific to Vicuna.

Includes: `cross_model_compare.py`, `extract_vicuna_full_sequence.py`, `vicuna_diagnostic.py`,
`vicuna_imbalance_test.py`, `vicuna_overfitting_test.py`, `vicuna_param_sweep.py`. Kept for provenance.
