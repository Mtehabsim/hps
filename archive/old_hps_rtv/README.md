# old_hps_rtv — early HPS + RTV-comparison scripts

Early HPS pipeline and comparisons against RTV (Representation Trajectory Verification) and related
baselines, plus layer-finding and ensemble scratch. Superseded by `src/` (`hps_core.py`,
`statistical_tests.py`, `rpf_on_cache.py`).

Includes: `hps_llama3.py`, `hps_rtv_inspired.py`, `rtv_standalone.py`, `rtv_find_layers.py`,
`hyperbolic_vs_euclidean_diverse.py`, `diagnostic_hps_vs_euc.py`, `attack_ensemble.py`,
`anthropic_mean_token_probe.py`. Historical only.
