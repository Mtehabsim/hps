# analysis_old — superseded analysis & diagnostic scripts

One-off analyses and confound checks from the negative-result arc, superseded by the consolidated
scripts in `src/` (e.g. `statistical_tests.py`, `hyperbolic_retrieval.py`).

Includes: curvature analyses (`analyze_curvature.py`, `analyze_sectional_curvature.py`,
`analyze_radial_severity.py`, `per_prompt_hyperbolicity.py`), confound diagnostics
(`diagnose_norm_confound.py`, `norm_check_diverse.py`, `investigate_contamination.py`,
`verify_saturation.py`), feature/probe ablations (`feature_ablation.py`, `control_experiments.py`,
`adversarial_compare.py`, `prediction_agreement.py`), and assorted scratch (`chat.py`,
`ws6_replot_controls.py`). Kept for provenance; not part of the active pipeline.
