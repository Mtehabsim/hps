# loose-data — root-level datasets from the DGX

Datasets that were dumped at the repo root by an old DGX commit and are **not referenced by any
current `src/` script**: `data_harmful_100.csv`, `data_harmless_100.csv`, `data_harmless_6500.csv`,
`llama3_attacks.json`, `llama3_attacks_clean.json`, `llama3_attacks_clean.report.json`. Kept as data
(not deleted) in case a future run needs them. (`validated_datasets.json`, which IS used, was moved to
`src/` instead.)
